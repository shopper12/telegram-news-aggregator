# 구현 메모

## 현재 포함된 기능

- Telethon 기반 텔레그램 채널 메시지 수집
- SQLite 저장
- 중복 뉴스 제거
- 섹터/키워드/티커 추출
- Markdown 리포트 생성
- 텔레그램 봇 발송 옵션
- Docker 실행 옵션
- 기본 테스트

## 아직 붙이지 않은 기능

1. 실시간 가격 검증
   - KRX/네이버/증권사 API 또는 Yahoo Finance 등 별도 연결 필요
   - 업비트/바이낸스 API는 코인용으로 별도 붙이는 게 맞음

2. 수급/거래량 검증
   - 한국 주식은 투자자별 매매동향, 거래대금, 종목별 프로그램 수급 등 별도 데이터 필요
   - 코인은 거래대금, OI, 펀딩비, ETF flow를 별도 모듈로 분리 권장

3. 스케줄러
   - Windows 작업 스케줄러
   - cron
   - GitHub Actions는 텔레그램 세션 파일 관리 때문에 비추천
   - EC2에서 systemd timer 권장

## 다음 개발 순서

1. config/channels.yaml 실제 채널 목록 입력
2. 1회 실행으로 수집 확인
3. 중복 제거 품질 확인
4. 섹터 키워드 사전 보강
5. 가격/거래량 API 연결
6. 최종 매매 후보 점수화
